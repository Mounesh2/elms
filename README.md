# 🎓 LMS - Learning Management System

Welcome to the **LMS (Learning Management System)** repository! This project is a comprehensive, feature-rich education platform built with modern web technologies.

---

## 🛠️ Tech Stack & Tools

This project leverages a powerful suite of tools to provide a seamless experience for both students and instructors.

### 🏗️ Core Framework
- **[Next.js 14](https://nextjs.org/)** - The React Framework for the Web. 🚀
- **[React 18](https://reactjs.org/)** - For building dynamic user interfaces. ⚛️
- **[TypeScript](https://www.typescriptlang.org/)** - Static type checking for safer code. 📘

### 🎨 Styling & UI
- **[Tailwind CSS](https://tailwindcss.com/)** - Utility-first CSS framework. 💅
- **[Framer Motion](https://www.framer.com/motion/)** - Smooth animations and transitions. ✨
- **[Lucide React](https://lucide.dev/)** - Beautiful & consistent icons. 🎨
- **[Radix UI](https://www.radix-ui.com/)** - Unstyled, accessible UI components. 🏗️

### 🗄️ Database & Backend
- **[PostgreSQL](https://www.postgresql.org/)** - Powerful relational database. 🐘
- **[Prisma](https://www.prisma.io/)** - Next-generation ORM. 💎
- **[Supabase](https://supabase.com/)** - Backend-as-a-Service for Auth & Storage. ⚡
- **[Drizzle ORM](https://orm.drizzle.team/)** - Lightweight and performant TypeScript ORM. 🏎️

### 💳 Payments & Services
- **[Stripe](https://stripe.com/)** - Global payment processing. 💳
- **[Cloudflare R2](https://www.cloudflare.com/products/r2/)** - S3-compatible object storage. ☁️
- **[Resend](https://resend.com/)** - Modern email delivery service. 📧

### 🤖 Artificial Intelligence
- **[Google Gemini AI](https://ai.google.dev/)** - Advanced AI capabilities. 🧠
- **[Groq AI](https://groq.com/)** - High-performance AI inference. ⚡

### 📦 Utilities
- **[Zod](https://zod.dev/)** - TypeScript-first schema validation. ✅
- **[React Hook Form](https://react-hook-form.com/)** - Performant form handling. 📝
- **[React Hot Toast](https://react-hot-toast.com/)** - Elegant notification toasts. 🍞
- **[Lucide React](https://lucide.dev/)** - Icon library. 🔍

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Mounesh2/LMS.git
   ```

2. **Navigate to the app directory:**
   ```bash
   cd learnhub
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev -- -p 3004
   ```
